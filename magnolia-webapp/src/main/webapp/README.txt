The basic Magnolia webapp on which projects can depend.

This package contains:

   magnolia-core 4.5.2
   magnolia-gui 4.5.2
   magnolia-jaas 4.5.2
   magnolia-module-admininterface 4.5.2
   magnolia-module-cache 4.5.2
   magnolia-module-exchange-simple 4.5.2
   magnolia-module-fckeditor 4.5.2
   magnolia-module-mail 4.5.2
   magnolia-module-store-client 4.5.2
   magnolia-rendering 4.5.2
   magnolia-templating 4.5.2
   magnolia-templating-editor 4.5.2
